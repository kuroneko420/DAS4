# DAS4

## Instructions for running the server and client:

```
1. Extract the folder from the archive
2. Enter into the directory that was just created
3. Open a console window and type and enter “java *.java”
4. Type and enter “rmiregistry &” (If you’re on Windows type “start rmiregistry”)
5. Type and enter java AuctionServer
6. Open a new console window, type and enter “java AuctionClient <name>”
7. (Optional) To run the server load test, open a new console window, type and enter 
“java MultiThreadStressServerTester”
```
